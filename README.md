# Sample Superstore Report: Executive Profitability Dashboard

## 📊 Project Overview
This project involves a comprehensive **Diagnostic Analysis** of the classic "Sample Superstore" dataset. I focused on building a clean, intuitive **Executive Dashboard** that moves beyond basic sales tracking to uncover hidden **profitability leakages** and logistical bottlenecks across North America.

## 🚀 Key Features & Functionality
* **Dynamic Slicing:** A City-level Slicer allows users to immediately filter the entire report for localized "deep-dive" analysis.
* **Profit Benchmarking:** Utilized **Gauge Visuals** for Profit, Quantity, and Sales to set clear high-level target benchmarks for the quarter.
* **Trend Analysis:** Developed a time-series line chart to visualize the regional profit journey (specifically noting the strong Q1 performance in the West).
* **Segment Breakdown:** Implemented interconnected Pie Charts for Segment, Sub-Category, and Category, revealing that **Copiers** are a key profit driver, primarily serving the **Technology** sector.

## 📈 Business Insights Derived (From the visualization)
* **Western Region Dominance:** The West outperforms all other regions in profit, indicating a strong market or lower operational costs that should be modeled elsewhere.
* **Profitable Sub-Categories:** Copiers, Phones, and Accessories show high margin potential, suggesting these categories should receive prioritized marketing spend.
* **Risk Identification:** Specific cities can be filtered to identify where small losses might be dragging down regional averages, helping logistics teams focus their optimization efforts.

## 🛠️ Tools Used
* **Power BI Desktop:** Report Development & Visual Styling (using custom JSON themes).
* **Power Query:** ETL processing and data shaping.
* **DAX Measures:** Calculated custom KPIs (Profit, Quantity, and Sales gauges).
* **Statistics:** Descriptive Analytics, outlier detection, and time-series modeling.
